# Terra -- Product Requirements Specification (PRS) v1.0

## 1. Product Vision

Terra is a satellite-led ecological intelligence platform providing
structural and functional biodiversity insights across landscape
portfolios.

## 2. Target Users

• Land for Wildlife property owners

• Bushcare groups

• Catchment organisations

• Corporate sponsors (portfolio view)

## 3. Core Requirements (Phase 1)

• Plot boundary interface

• Monthly ecological composite generation

• Regional ecosystem identification

• Mission recommendation engine

• Basic reporting export

## 4. Non-Functional Requirements

• Monthly processing cadence

• Cloud-native storage architecture

• Clear tier transparency (Tier 1 vs Tier 2)

• Scalable to 25 km² and beyond

## 5. Future Requirements (Phase 2+)

• Acoustic integration

• Portfolio HHI scoring

• Elite eligibility tracking

• External validation workflows
